Projeto Logística Solidária

Sistema web desenvolvido em Django para gerenciamento de doações, pedidos, rastreamento e avaliações.

========================================
REQUISITOS
========================================

- Python 3.12+
- Git
- Pip

========================================
CLONANDO O PROJETO
========================================

git clone https://github.com/pedromorais4709-arch/Projeto_logistica_solidaria.git

========================================
ACESSANDO A PASTA
========================================

cd Projeto_logistica_solidaria

========================================
CRIANDO AMBIENTE VIRTUAL
========================================

Linux:

python -m venv .venv
source .venv/bin/activate

Windows:

python -m venv .venv
.venv\Scripts\activate

========================================
INSTALANDO DEPENDÊNCIAS
========================================

pip install -r requirements.txt

========================================
APLICANDO MIGRATIONS
========================================

python manage.py makemigrations
python manage.py migrate

========================================
EXECUTANDO O PROJETO
========================================

python manage.py runserver

O sistema estará disponível em:

http://127.0.0.1:8000/

========================================
FUNCIONALIDADES
========================================

- Cadastro de usuários
- Login e autenticação
- Registro de doações
- Criação de pedidos
